# Fashion MNIST Image Classification using CNN

## Project Overview
This project uses a Convolutional Neural Network (CNN) built with TensorFlow to classify clothing images from the Fashion MNIST dataset into 10 categories.

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib

## Dataset
Fashion MNIST Dataset

## Model
- CNN
- Conv2D
- MaxPooling2D
- Dense Layers
- Softmax Output
